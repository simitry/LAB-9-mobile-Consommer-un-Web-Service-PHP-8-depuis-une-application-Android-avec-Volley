# PHP Web Service - school1

Copier le dossier `server/projet` vers :

```text
C:\xampp\htdocs\projet
```

Créer la base et les données de test avec :

```sql
server/projet/sql/school1.sql
```

URLs utilisées par Android :

```text
http://10.0.2.2/projet/ws/createEtudiant.php
http://10.0.2.2/projet/ws/loadEtudiant.php
http://10.0.2.2/projet/ws/updateEtudiant.php
http://10.0.2.2/projet/ws/deleteEtudiant.php
```

Depuis un navigateur sur Windows, l'équivalent est :

```text
http://localhost/projet/ws/loadEtudiant.php
```
